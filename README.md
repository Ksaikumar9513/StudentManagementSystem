# 🎓 Student Management System 🎓 

A frontend-only live demo of the **Student Management System** that simulates the complete student management workflow. The application is designed for deployment on static hosting platforms like **Netlify** and **GitHub Pages**, allowing recruiters to explore the interface without requiring a backend server.

> **Note:** This demo uses **localStorage** and **sessionStorage** to simulate database operations. The original backend implementation is built using **Core Java, Servlets, JDBC, and Oracle SQL**.

## 📖 Project Overview

The Student Management System provides an interface for managing students, courses, attendance, and marks. This live version demonstrates the application's workflow using browser storage, making it easy to showcase without setting up a server or database.

## 🔄 How the Demo Works

Instead of connecting to a real database, this project stores data in:

- localStorage
- sessionStorage

This makes the application fully functional on static hosting platforms while simulating real CRUD operations.
